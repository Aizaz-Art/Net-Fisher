## Requirement
- python 3
- pip3
- scapy
- mac_vendor_lookup
- prettytable
## Installation

Copy all the commands and paste on Terminal.

```bash
  git clone https://github.com/Crack-Pathan/Net-Fisher.git
  cd Net-Fisher
  chmod u+x Net-Fisher.py
  pip3 install -r requirements.txt
  python Net-Fisher.py
```
